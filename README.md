# Life System

A journaling and life planning system powered by [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Setup

1. [Install Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview)
2. Clone and start:
   ```
   git clone https://github.com/rushatgabhane/life.git
   cd life
   claude
   ```
3. Ask Claude to help you fill in `plan.md` and `journal/2026/goals.md`

## Usage

- Say **"morning"** to start your day (creates journal, reviews goals, plans priorities)
- Say **"evening"** to reflect on the day
- Journal entries live in `journal/YYYY/MM/YYYY-MM-DD.md`
